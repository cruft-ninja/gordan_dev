# AI Habitat Glossary

| Term | Definition |
| :--- | :--- |
| **Habitat** | The root development environment (`~/dev`). |
| **Directives** | Global laws (`AI_SECURITY_POLICY.md`, `AI_DOCS_STANDARD.md`). |
| **Cores** | Project-specific `gemini.md` files. |
| **Hats** | Specialized personas or expert roles. |
| **Hot-swapping** | Loading a new **Core** mid-session via `read_file`. |
| **The Ledger** | The collection of session logs in `./docs/` subdirectories. |
| **The Window** | The finite token limit (context window) of the current session. |
