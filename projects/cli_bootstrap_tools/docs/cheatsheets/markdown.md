# Markdown Cheatsheet

Markdown is used for `.md` files (like this one) to format text nicely.

## 1. Headers
```markdown
# H1 - Main Title
## H2 - Section
### H3 - Subsection
```

## 2. Text Formatting
| Style | Syntax | Example |
| :--- | :--- | :--- |
| **Bold** | `**text**` | **Bold** |
| *Italic* | `*text*` | *Italic* |
| `Code` | `` `text` `` | `Code` |
| ~~Strike~~ | `~~text~~` | ~~Strike~~ |

## 3. Lists
**Bulleted List:**
```markdown
- Item A
- Item B
  - Sub-item
```

**Numbered List:**
```markdown
1. First step
2. Second step
```

## 4. Links & Images
```markdown
[Link Text](https://example.com)

![Image Description](image.png)
```

## 5. Code Blocks
Use three backticks (\`\`\`) for multi-line code.
\```python
print("Hello")
\```
